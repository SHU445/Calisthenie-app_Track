## 📸 OBSERVATIONS VISUELLES DES SCREENSHOTS

### Interface Générale
- Ambiance : Sombre, martiale, militaire, sobre et professionnelle
- Style : Design system moderne avec typographie impactante
- Contraste : Excellent entre le fond sombre et les accents rouges
- Lisibilité : Optimale avec texte off-white sur fond très sombre

### Navigation
- Fond : Gradient horizontal gunmetal avec backdrop blur
- Logo : Cercle rouge (#9E1B1B) avec icône de livre/dossier blanc
- Texte logo : "Callisthénie Tracker" en rouge (#9E1B1B) et gris clair
- Liens : Gris clair avec icônes, hover probable en rouge
- Boutons action : "CONNEXION" (bordure), "INSCRIPTION" (fond rouge)

### Page d'accueil (Hero)
- Titre principal : "CALLISTHÉNIE TRACKER" 
  - "CALLISTHÉNIE" en rouge (#9E1B1B)
  - "TRACKER" en blanc off-white (#DFD9CB)
- Typographie : Bebas Neue (Display font) très impactante
- Boutons CTA : Rouge vif avec ombre lumineuse rouge
- Statistiques : Cercles rouges avec icônes + chiffres blancs

### Cartes (Cards)
- Fond : Gradient gunmetal (#13171A vers #0B0F12)
- Bordure : Subtile gris-acier semi-transparent
- Ombre : Ombre sombre avec légère teinte
- Icônes : Couleurs variées (doré, bleu-vert, rouge)
- Hover : Bordure rouge avec glow effect

### Badges de Rangs (System de Difficulté)
Observés sur la page "Système de Rangs" :

**F (Rang F - Base)** : 
- Badge : Cercle VERT (#10B981 ou similaire vert émeraude/militaire)
- Style : Lettres blanches sur fond vert
- Contexte : "Niveau de base - Immédiat"

**D (Rang D - Débutant)** :
- Badge : Cercle CYAN/TURQUOISE (#47B7CC ou bleu-cyan clair)
- Style : Lettres blanches sur fond cyan
- Contexte : "Niveau débutant - 1 à 2 mois"

**C (Rang C - Débutant avancé)** :
- Badge : Cercle BLEU (#2E79C5 ou bleu moyen)
- Style : Lettres blanches sur fond bleu
- Contexte : "Niveau débutant avancé - 2 à 6 mois"

**B (Rang B - Intermédiaire)** :
- Badge : Cercle VIOLET (#7B3FA5 ou violet moyen)
- Style : Lettres blanches sur fond violet
- Contexte : "Niveau intermédiaire - 6 à 15 mois"

**A (Rang A - Intermédiaire supérieur)** :
- Badge : Cercle ORANGE (#F59E0B ou ambre/orange)
- Style : Lettres blanches/noires sur fond orange
- Contexte : "Intermédiaire supérieur - 15 à 36 mois"

**S (Rang S - Avancé)** :
- Badge : Cercle ROUGE (#EF4444 ou #9E1B1B rouge vif)
- Style : Lettres blanches sur fond rouge
- Contexte : "Avancé - 3 à 5 ans"

**SS (Rang SS - Élite)** :
- Badge : Cercle VIOLET/LAVANDE (#B788E8 ou violet clair)
- Style : Lettres noires sur fond lavande
- Contexte : "Élite - Jusqu'à 5 ans+"

### Formulaires (Page Login)
- Container : Card sombre avec bordure subtile
- Champs input : Fond gunmetal (#13171A) avec bordure gris-acier
- Icônes : Gris clair dans les inputs (utilisateur, cadenas)
- Labels : Texte gris clair off-white
- Placeholder : Gris moyen steel (#AEB6BD)
- Bouton principal : Rouge crimson pleine largeur
- Liens : Texte rouge pour "S'inscrire gratuitement"
- Encadré démo : Fond très légèrement différent

### Footer
- Fond : Semi-transparent avec backdrop blur
- Logo : Cercle rouge identique à la navigation
- Texte : Gris clair (#AEB6BD ou #BCC4CB)
- Sections : Navigation, À propos
- Copyright : Gris plus sombre
- Liens : Hover probable en rouge

# │ 🔄 SYSTÈME DE THÈMES                                                 │
# └─────────────────────────────────────────────────────────────────────┘

## === IMPLÉMENTATION ===

**Store Zustand** (`themeStore.ts`) :
- État : 'dark' (Martial) | 'violet' (Gothic)
- Persistance : localStorage
- Fonction : `setTheme()`, `toggleTheme()`

**Provider** (`ThemeProvider.tsx`) :
- Applique classe `.violet` sur `<html>` si mode violet
- Effet de synchronisation avec le store
- Détection initiale depuis localStorage

**Configuration** (`tailwind.config.js`) :
- Palette `martial.*` pour le thème sombre
- Palette `gothic.*` pour le thème violet
- Alias `sport.*` pour rétrocompatibilité

**Styles** (`globals.css`) :
- Règles de base pour `.dark` (par défaut)
- Surcharges `.violet` pour tous les composants
- Transitions automatiques entre thèmes

## === ACTIVATION DU THÈME ===

**Via page Paramètres** (`/parametres`) :
1. Sélection "Mode Sombre" ou "Mode Violet"
2. Sauvegarde dans localStorage ('theme')
3. Application immédiate via store
4. Classe CSS ajoutée/retirée dynamiquement

## === CLASSES AVEC SUPPORT DUAL-THEME ===

Toutes les classes `martial-*` et `sport-*` s'adaptent :
- Mode dark : Couleurs martial (rouge/gris)
- Mode violet : Couleurs gothic (violet/lavande)
- Transition fluide entre les deux
