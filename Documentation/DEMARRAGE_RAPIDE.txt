📋 ÉTAPES À SUIVRE (dans l'ordre) :
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  CONFIGURER LA BASE DE DONNÉES
   ┌─────────────────────────────────────────────────────────────────────┐
   │ a) Créez une base PostgreSQL gratuite sur https://neon.tech/       │
   │ b) Configurez Prisma Accelerate sur https://cloud.prisma.io/       │
   │ c) Copiez les 2 URLs dans .env.local :                             │
   │                                                                     │
   │    DATABASE_URL="prisma+postgres://accelerate..."                  │
   │    DIRECT_URL="postgresql://user:password@host..."                 │
   └─────────────────────────────────────────────────────────────────────┘

2️⃣  INITIALISER PRISMA
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Option A - Automatique (recommandé) :                              │
   │   Double-cliquez sur : setup-prisma.bat                            │
   │                                                                     │
   │ Option B - Manuel :                                                │
   │   npm run prisma:generate                                          │
   │   npm run prisma:push                                              │
   │   npm run prisma:seed                                              │
   └─────────────────────────────────────────────────────────────────────┘

3️⃣  (OPTIONNEL) MIGRER MONGODB
   ┌─────────────────────────────────────────────────────────────────────┐
   │ Si vous avez des données MongoDB à migrer :                        │
   │   npm run prisma:migrate                                           │
   └─────────────────────────────────────────────────────────────────────┘

4️⃣  TESTER L'APPLICATION
   ┌─────────────────────────────────────────────────────────────────────┐
   │   npm run dev                                                       │
   │                                                                     │
   │ Pour voir vos données visuellement :                               │
   │   npm run prisma:studio                                            │
   │   (ouvre http://localhost:5555)                                    │
   └─────────────────────────────────────────────────────────────────────┘

⚠️  PROBLÈMES FRÉQUENTS :
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Environment variable not found: DATABASE_URL"
   ➜ Vérifiez que .env.local contient DATABASE_URL

❌ "Can't reach database server"
   ➜ Vérifiez que votre base PostgreSQL est accessible

❌ "Prisma Client not generated"
   ➜ Exécutez : npm run prisma:generate


🌐 DÉPLOIEMENT SUR DENO DEPLOY :
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Une fois que tout fonctionne localement :

1. Poussez sur GitHub :
   git add .
   git commit -m "Migration vers Prisma"
   git push origin main

2. Configurez Deno Deploy (https://dash.deno.com/)
   - Créez un projet
   - Liez votre repo GitHub
   - Ajoutez DATABASE_URL et DIRECT_URL

3. Déployez automatiquement !

Consultez README_PRISMA.md pour les détails complets.
